# ChangeLog

## 0.9.0 (2020-02-17)

- 支持 Cocos Creator v2.3.0
  - 由于2.3.0和2.2.2，2.2.1差别较大，因此，此Tag的最新项目已经无法在2.2.1，2.2.2中重新打开，如果确实需要在 2.2.1，2.2.2 中打开，请先切换到旧版本的tag（`0.8.0`），才能用 2.2.1，2.2.2打开
